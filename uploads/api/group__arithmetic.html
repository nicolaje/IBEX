typedef TemplateDomain<Affine2> ibex::Affine2Domain

Affine Domain.

An affine domain is either:

  • an affine form (ibex::Affine2)
  • a vector of affine forms (ibex::Affine2Vector)
  • a matrix of affine forms (ibex::Affine2Matrix)
  • or an array of affine matrices (ibex::Affine2MatrixArray)
typedef TemplateDomain<Interval> ibex::Domain

Interval Domain.

An interval domain is either:

  • an interval (ibex::Interval)
  • a vector of intervals (ibex::IntervalVector)
  • a matrix of intervals (ibex::IntervalMatrix)
  • or an array of interval matrices (ibex::IntervalMatrixArray)

Generalized division.

The result of x/y is stored in out1 and out2. These intervals contain the lower and upper part respectively of the division. If the result of the generalized division is a single interval, out2 is set to the empty interval. If the result is the empty set, both out1 and out2 are set to the empty interval.

Note
Contrary to the "cset" theory, the result is empty if y=[0,0] (whatever x is).
