Detailed Description

Round-robin bisector.

Constructor & Destructor Documentation

ibex::RoundRobin::RoundRobin ( double  w = Bsc::default_prec,
double  ratio = Bsc::default_ratio 
)

Create a bisector with round-robin heuristic.

Parameters
w- the minimum width (diameter) an interval must have to be bisected (see w). Default value is 1e-07.
ratio(optional) - the ratio between the diameters of the left and the right parts of the bisected interval. Default value is 0.45.

int &  last_var 
)
virtual

Bisect a box.

Parameters
last_var- the last component that has been bisected (hence, if the domain of the last_var+1 component is wider than w, it will be bisected) or -1 if this is the first call (in this case, the variable inspected first is the first one). In case all the variables have domain diameter less than w, the first variable (0) will be bisected.

In return, last_var is set to the variable just bisected.

Member Data Documentation

const double ibex::RoundRobin::w

Precision.

Allow to skip variables whose domain is too small in the round-robin heuristic. This parameter usually matches the ceil parameter of the precision contractor since boxes that are too small are usually both not bisected and considered as precise enough.

const double ibex::RoundRobin::ratio

Ratio between the diameters of the left and right parts of a bisected interval.
