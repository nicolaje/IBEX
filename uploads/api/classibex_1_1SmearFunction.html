Create a bisector with Smear function heuristic,i.e using the impacts based on the Jacobian matrix. Abstract class with 4 implementations SmearMax, SmearSum, SmearMaxRelative, SmearSumRelative corresponding to different variants for computing the variable to be bisected, all variants using the Jacobian matrix.

Parameters
sys- The system of equations.
w(optional) - - the minimum width (diameter) an interval must have to be bisected (see w).
ratio(optional) - the ratio between the diameters of the left and the right parts of the bisected interval. Default value is 0.45
