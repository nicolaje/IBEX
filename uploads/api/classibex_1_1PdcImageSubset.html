Detailed Description

Test if a box is inside the range of a function over an (implicit) set.

Based on the algorithm by Goldsztejn & Jaulin. See p.12 of the paper "Inner Approximation of the Range of Vector-valued Functions" Reliable Computing, 2010.

- the function f must be "square" (from R^n to R^n).
