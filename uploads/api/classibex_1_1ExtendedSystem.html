Detailed Description

System where a goal is encoded as a constraint.

Constructor & Destructor Documentation

ibex::ExtendedSystem::ExtendedSystem ( const System &  sys)

Normalize sys and encode the goal function as a constraint.

The resulting system includes (n+1) variables, the n original variables x_1,...,x_n and an an additional "goal" variable y. The index of the goal variable is goal_var(). It also includes (m+1) constraints. The index of the constraint y-goal(x)=0 (where goal(x) is the goal function) is goal_ctr(). The others constraints are normalized copy of the original ones. The name of the goal variable is goal_name.

To avoid confusion, the extended system has no goal (but it could be "y").
