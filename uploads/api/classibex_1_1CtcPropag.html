Detailed Description

Propagation contractor.

This class is an implementation of the classical interval variant of the AC3 constraint propagation algorithm.

Constructor & Destructor Documentation

ibex::CtcPropag::CtcPropag ( const Array< Ctc > &  cl,
double  ratio = default_ratio,
bool  incr = false 
)

Create a AC3-like propagation with a list of contractors.

Parameters
cl- The list of contractors.
ratio(optional) - Criterion for stopping propagation. If a projection does not remove more that ratio times the diameter of a variable domain, then this reduction is not propagated. The default value is default_ratio.
incr(optional) - Whether the propagation works incrementally. This parameter is only used when contraction is called with an "impact" bool mask.
See Also
contract(IntervalVector&, const BoolMask&).

Member Function Documentation

virtual void ibex::CtcPropag::contract ( IntervalVector &  box)
virtual

Enforces propagation (e.g.: HC4 or BOX) fitering.

Call contract(IntervalVector&, const BoolMask&) with the mask set to all the variables. If incremental is true, the propagation will start from the impacted variables only (instead of from all the variables).
