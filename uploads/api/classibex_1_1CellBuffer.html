Detailed Description

Cell Buffer.

This class allows to control the order in which cells are treated by the paver, i.e., to control the way domains are explored. For instance, if the container is implemented as a stack, the paver will perform a depth-first search. If it is a file, the paver will perform a breadth-first search, and so on.

int ibex::CellBuffer::capacity

Limit the buffer size.
