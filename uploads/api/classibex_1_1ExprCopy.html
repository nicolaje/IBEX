Duplicate y[i], where y is an expression.

If y is a vector, the ith argument of y is copied, i.e., copy-of(y[i]) instead of returning (copy-of(y))[i].

Duplicate y[i][j], where y is an expression.

If y is a matrix, a copy of the (i,j) entry of y is returned, i.e., copy-of(y[i][j]). Else if y is a vector, a copy-of(y[i])[j] is returned. Otherwise, the returned expression is (copy-of(y))[i][j].
