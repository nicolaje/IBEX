Detailed Description

Operator-specific backtrackable data.

A backtrackable is a structure propagated through the search tree. It is therefore only used by strategies. Each node in the search tree has its own structure, this structure being inherited from the father node. The mechanism of inheritance can be controlled thanks to the down function. For the moment, control can only be made downward: there is no way of updating a node structure by aggregating children node structures when backtracking (this might be done in a future release).

This class is an interface to be implemented by any operator data class associated to a cell.
