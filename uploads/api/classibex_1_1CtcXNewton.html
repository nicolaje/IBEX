Detailed Description

X_Newton contractor.

This class is an implementation of the X-Newton algorithm

Creates the X_Newton contractor.

Parameters
sys- The system (extended or not).
cpoints- The vector of corner selection in linearization (X_INF, X_SUP, RANDOM, RANDOM_INV)
cmode- X_NEWTON (contracts all the box) | LOWER_BOUNDING (in optimization only improves the left bound of the variable y)
lmode- TAYLOR | HANSEN : linear relaxation method.
max_iter_soplex- The maximum number of iterations for Soplex (default value 100)
max_diam_deriv- The maximum diameter of the derivatives for calling Soplex (default value 1.e5)
max_diam_box- The maximum diameter of the variables for calling Soplex (default value 1.e4) Soplex may lose solutions when it is called with "big" domains.
