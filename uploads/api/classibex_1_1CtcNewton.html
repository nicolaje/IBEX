Parameters
ceil- Criterion for applying interval Newton contractor. When the diameter of all components of the current box is smaller than ceil, the interval Newton method is applied. This criterion is useful to avoid useless computations of the Jacobian matrix for wide boxes.

Default value is default_ceil.
