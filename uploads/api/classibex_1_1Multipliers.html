void ibex::Multipliers::init_root ( int  m,
int  r,
int  k 
)

Set the number of Lagrange Multiplier to 1+m+r+k.

m corresponds to the number of inequalities, r to the number of equalities, k the number of bound constraints. Set the domain of the "special" multiplier u to [0,1], the domain for the m+k inequalities/bound constraints to [0,1] and the domain for the r equalities to [-1,1].
