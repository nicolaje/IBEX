Detailed Description

Generic bisector.

Base class of all bisectors. A bisector is an operator used by a strategy (like the solver ) to make a choice point. An implementation of this class must provide a bisect(const IntervalVector& box) function.

virtual std::pair<IntervalVector,IntervalVector> ibex::Bsc::bisect ( Cell &  cell)
virtual

Bisect the current cell and return the result.

The information in a cell is e.g., used to get the last bisected variable in case the bisector is called by a solver/paver (e.g., RoundRobin). Implementation is optional. By default, this function call bisect(cell.box). See bisect(const IntervalVector&).

Reimplemented in ibex::RoundRobin.
