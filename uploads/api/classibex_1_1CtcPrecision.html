Detailed Description

Precision contractor.

Typically used as a stop criterion for pavers. This contractor controls the precision of boxes of a paver: when the interval diameter of all the variables in the current box is lower than ceil , the box is entirely contracted (ibex::EmptyBoxException). The boundary of the set to be described by the paver is therefore usually associated to the boxes this contractor has removed from the search. In this case, the precision of the boundary is directly related to the parameter ceil .
