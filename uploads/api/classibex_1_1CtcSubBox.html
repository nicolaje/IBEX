Detailed Description

Apply a contractor on a sub-box.

The input vector x of this contractor is decomposed into two sub-vectors y and z. The contractor c is then applied on y. Components of z are not impacted.

Parameters
c- The contractor to apply
mask- tells which components belong to y, the subbox given in argument to c.

Example:
Assume c.nb_var=5 and mask=(0,1,1,0,1).
Then y=(x_2,x_3,x_5) and z=(x_1,x_4).
