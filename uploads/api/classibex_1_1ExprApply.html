Detailed Description

Function application expression.

In the following Quimper example:

Variables
a in [0,10];
function z=foo(x,y)
z=sqrt(x^2+y^2);
end;
Constraints
foo(a,1-a)=1
...

f(a,1-a) is an instance of ExprApply, where func is "foo" and the actual arguments arg(0) and arg(1) are the subexpressions a and 1-a.
