Parameters
sysThe system (the extended system in case of optimization)
cpointsThe vector of corner selection in linearization (X_INF, X_SUP, RANDOM, RANDOM_INV)
cmodeX_NEWTON (contracts all the box) | LOWER_BOUNDING (in optimization only improves the left bound of the variable y)
lmodeAFFINE2 | TAYLOR | HANSEN | COMPO: linear relaxation method.
max_iter_soplex: the maximum number of iterations for Soplex (default value 100)
max_diam_deriv: the maximum diameter of the derivatives for calling Soplex (default value 1.e5)
max_diam_box: the maximum diameter of the variables for calling Soplex (default value 1.e4) Soplex may lose solutions when it is called with "big" domains.
