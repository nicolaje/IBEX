ibex::IntervalMatrix::IntervalMatrix ( int  m,
int  n,
double  x[][2] 
)

Create an interval matrix from an array of doubles.

Create the IntervalMatrix:
[x[0][0],x[0][1]] ; ... ; [x[n-1][0], x[n-1][1]]
[x[n][0],x[n][1]] ; ... ; [x[2n-1][0],x[2n-1][1]]
...
[x[(m-1)n][0],x[n][1]] ; ... ; [x[mn-1][0],x[2n-1][1]]

Parameters
boundsan (mxn)x2 array of doubles
Precondition
m>0, n>0
