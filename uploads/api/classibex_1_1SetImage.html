Detailed Description

Set image using Goldsztejn-Jaulin algorithm (Reliable Computing, 2010).

Based on the algorithm by Goldsztejn & Jaulin. See "Inner Approximation of the Range of Vector-valued Functions" Reliable Computing, 2010.

Note: the bisection procedure is fixed to LargestFirst.

Parameters
f- A function from R^n to R^n
x- The box in IR^n
